## Hi, I'm Abhijeet Mohanan 👋

I am a seasoned **Infrastructure and Platform Engineer** with a deep passion for building robust, scalable, and highly available systems. I bridge the gap between software engineering and system administration, focusing on automation, reliability, and security at scale.

### 🛠️ Core Expertise

My technical focus lies in designing and managing modern cloud-native architectures:

*   **Kubernetes & Container Orchestration:** Deep expertise in deploying, scaling, and managing complex K8s clusters (EKS, self-hosted). Proficient with Helm, operators, and cluster networking (CNI).
*   **Cloud Infrastructure (AWS):** Extensive experience architecting on Amazon Web Services. I leverage core services (VPC, EC2, S3, RDS, IAM, EKS) to build secure and cost-effective environments.
*   **Advanced Networking:** Solid understanding of TCP/IP, BGP, routing, load balancing, VPC peering, and Transit Gateways. I build networks that don't fail when you need them most.
*   **Infrastructure as Code (IaC):** Heavy automation user. I treat infrastructure as software using Terraform, CloudFormation, and Ansible.
*   **CI/CD & DevOps:** Building smooth developer experiences and automated pipelines using GitHub Actions, GitLab CI, or ArgoCD for GitOps workflows.

### 🚀 What I Do

I don't just spin up servers; I engineer platforms. My goals are always:
1.  **Resilience:** Building self-healing systems that can survive region outages.
2.  **Developer Velocity:** Creating platforms that allow developers to ship code safely and quickly without worrying about the underlying plumbing.
3.  **Observability:** Implementing monitoring and logging (Prometheus, Grafana, ELK/Datadog) so we know a system is breaking before the users do.
